## Overview

BBEdit Professional Suite is a complete professional text processing platform that serves developers, writers, and data professionals with the project management, version control integration, and automation depth needed for production-scale text work. It extends the core editor with features that handle the organizational and collaborative demands of professional text workflows.

Project management organizes related files into project windows with file browsers, build integration, and per-project settings that keep multi-file work organized. The project browser supports filtering, sorting, and search that navigates large project structures efficiently. SCM integration shows version control status alongside the file listing, providing project-level awareness of uncommitted changes.

Live preview renders HTML and Markdown documents in a synchronized preview pane that updates as the source is edited, enabling visual feedback for web content and documentation authoring without switching to a browser. The preview respects CSS styling, and the synchronization scrolls to the corresponding position in both panes. This capability makes BBEdit practical for web authoring and documentation writing.

Shell worksheet integration embeds terminal functionality within the editor, executing commands and capturing output in an editable text document that combines the command line and text editor workflows. The worksheet enables interactive exploration, log analysis, and command development with full editing capability applied to terminal output. Multiple worksheets can run simultaneously for different tasks.

BBEdit Professional Suite is the text processing platform for professionals who need the combination of editing power, text transformation capability, and project organization that three decades of focused macOS development have produced. Its staying power reflects genuine, irreplaceable capability that no alternative has succeeded in matching.

## Performance & Integration

BBEdit is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while BBEdit processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
